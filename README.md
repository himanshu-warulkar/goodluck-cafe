# ☕ Goodluck Cafe — Digital Experience Platform

A modern, full-stack web platform built for **Goodluck Cafe Pvt. Ltd. (Est. 1935, Pune)** — blending vintage charm with modern digital experiences.

This project showcases a **scalable SaaS-style architecture** for cafes, enabling menu display, customer engagement, and future automation capabilities.

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/goodluck-cafe.git
cd goodluck-cafe
```

---

## 🖥️ Run Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## ⚙️ Run Backend

```bash
cd backend
npm install
npm run dev
```

---

## 🔨 Build for Production

### Backend

```bash
npm run build
npm start
```

---

## 🌐 Deployment

### Frontend (Vercel / Netlify)

* Root directory: `frontend`
* Build command: `npm run build`
* Output: `dist`

---

### Backend (Render)

* Root directory: `backend`
* Build: `npm install && npm run build`
* Start: `npm start`
